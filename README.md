the project of software-engineering course

可行性分析报告地址

本项目分为一个前端项目和两个后端项目

* 前端1：使用vue3+ts+elementplus+router+vuex+axios实现博客和交易平台前端界面以及前端网页请求
* 后端1：Nodejs_backend:负责交易平台的后端接口
* 后端2：python实现搜索和推荐功能


代码提交流程：
* 将仓库fork到自己仓库中
* 然后克隆到本地进行开发
* 开发后commit提交并push到远程仓库
* 然后提交PR，我会合并到我的仓库中(项目主仓库)
